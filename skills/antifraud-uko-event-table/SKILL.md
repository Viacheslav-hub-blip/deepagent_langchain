# Не карточные события: `csp_afpc_sss_inc.uko_event`

События вне специализированного карточного контура (UKO).

**Когда:** ДБО, ВСП, СБОЛ, СБП, переводы, сервисные операции, отделение, счета/вклады, смена профиля, подтверждения — по `event_channel`, `surface`, `type_operation` или смыслу задачи.

**Связь с hits:** как у карточных — `event_id` при наличии; иначе клиент + дата/время + сумма + тип + реквизиты/получатель при необходимости.

**Не путать:** не подменять карточные кейсы; строка не равна сработке; события могут отсутствовать в hits.
