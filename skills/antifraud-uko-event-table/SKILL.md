# UKO Event Table

Описание: точка входа для не карточной raw-истории.

Источник: `csp_afpc_sss_inc.uko_event`.

Используй для ДБО, СБОЛ, СБП, ВСП, переводов, счетов, вкладов, сервисных действий, профиля клиента, подтверждений и событий отделения.

Открой `../_shared/data-sources.md` для ключей, полей и правил связывания.

Не используй для карточных сценариев. Для `policy_action`, правил, резолюций и save-флагов возвращайся в hits.
