# Terms Glossary

Описание: минимальный словарь терминов; полные правила в `../_shared/antifraud-core.md` и `../_shared/data-sources.md`.

| Термин | Значение |
|---|---|
| Сработка | Операция, попавшая в hit-таблицу антифрода. |
| Hit table | Таблица только со сработками. |
| Raw event table | Полная история событий, включая операции без сработки. |
| Карточный канал | Карты, POS, e-commerce, ATM, авторизации, токены. |
| UKO | ДБО, ВСП, СБП, переводы и сервисные операции. |
| `policy_action` | Первичное решение антифрода. |
| Resolution | Итог подтверждения или расследования. |
| `is_save` | Флаг предотвращенного мошенничества. |
| CSI | Индекс клиентской удовлетворенности. |
| `epk_id` | Клиентский идентификатор для объединения источников. |
