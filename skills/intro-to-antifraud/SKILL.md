# Intro to Antifraud

Описание: точка входа для общих вопросов об антифроде.

Открой при необходимости:

- `../_shared/antifraud-core.md` - бизнес-логика, резолюции, подтверждение, приватность.
- `../_shared/data-sources.md` - различие hits/raw и выбор источника.

Фокус ответа:

- операция -> признаки -> правила/модели -> `policy_action` -> подтверждение -> резолюция;
- `policy_action`, резолюция и факт проведения операции всегда разделяются;
- для юрлиц и эквайринга явно отмечаются отдельные ограничения;
- для таблиц, KPI или паттернов переходи к профильному skill.
