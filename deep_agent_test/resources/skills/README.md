# Skills

Этот каталог содержит основной рабочий набор доменных знаний агента.

Принцип:

- `SKILL.md` содержит короткую карточку источника или workflow.
- Полные списки полей и подробные правила связи лежат рядом в `fields.md` и `joins.md`.
- Middleware предзагружает только `SKILL.md`.
- Если модели не хватает информации, она читает дополнительный файл через filesystem tools, например `/skills/hit-table/fields.md`.

Такой формат держит базовый prompt компактным: в контекст сразу попадают только
правила выбора источника, ключевые поля и критические ограничения. Подробные списки
полей загружаются по необходимости.

## Структура

```text
resources/skills/
  hit-table/
    SKILL.md
    fields.md
    joins.md
  cards-event-table/
    SKILL.md
    fields.md
  uko-event-table/
    SKILL.md
    fields.md
```

## Правило добавления знаний

- В `SKILL.md` добавляй только то, что нужно для маршрутизации, выбора таблицы,
  ключей связи и базового набора полей.
- В `fields.md` выноси полный список полей и описания редких колонок.
- В `joins.md` выноси подробные правила связи источников.
- Workflow-skills описывают алгоритм анализа и ссылаются на карточки таблиц.
