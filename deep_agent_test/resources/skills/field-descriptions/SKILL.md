# Индекс справочников полей

Этот skill не содержит полный справочник полей. Он нужен только для маршрутизации к подробным файлам.

Не загружай полный справочник заранее. Сначала используй краткие карточки таблиц:

- `/skills/hit-table/SKILL.md`
- `/skills/cards-event-table/SKILL.md`
- `/skills/uko-event-table/SKILL.md`

Если нужен полный список полей, прочитай один конкретный файл:

- `/skills/hit-table/fields.md` - поля `hits`.
- `/skills/cards-event-table/fields.md` - поля `cards`.
- `/skills/uko-event-table/fields.md` - поля `uko`.

Когда читать справочник:

- пользователь спрашивает значение конкретного поля;
- нужно выбрать редкие поля для выгрузки;
- `load_data` вернул schema error;
- нужно проверить, в какой таблице находится поле.

Не читай все `fields.md` сразу. Выбирай файл по таблице, которая нужна для текущего шага.
